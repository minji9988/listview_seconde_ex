# 앱 개요

<img width="30%" src="https://user-images.githubusercontent.com/104492622/227769553-2d9fd183-ada0-4c35-81d8-60760630ca44.gif"/>

- 리스트의 item 클릭 시 toast 메세지가 뜨도록 한다.
- 이때 toast 메세지는 data model의 첫 번째 요소만 뜨도록 한다.
예)         

List = [
     ListViewModel("a", "b"),
     ListViewModel("c", "d"),
     ListViewModel("e", "f"),
       ]

위와 같은 List가 있다면
a, c, e만 뜨도록 하는 것이다

</br>
